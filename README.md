## About The Project

### Project Description

The application will be connected to the game, Stardew Valley. The app will show some of the collectables within the game, from Items Shipped, Fish, Artefacts, Minerals and Cooking. This library system will also then show each categories information on a certain object.

For example, see the following:

#### Items Shipped
* Source
* Season
* Healing Effect
* Sell Price
* Villager Reactions
* Bundles
* Recipes
* Crafting

#### Fish
* Found in
* Time of Day
* Season
* Weather
* Difficulty
* Behaviour
* Villager Reactions
* Bundles
* Recipes

#### Artefacts
* Artefact Spots
* Monster Drops
* Other Sources
* Donation Reward
* Sell Price
* Villager Reactions

#### Minerals
* Source
* Sell Price
* Gemologist Profession Price
* Villager Reactions
* Bundles
* Recipes

#### Cooking
* Source
* Healing
* Sell Price
* Qi Seasoning
* Recipe Source
* Ingredients
* Villager Reactions
* Recipes
